# Benchmark Results

运行 `npm run benchmark` 查看性能对比。

## 测试场景

1. **阶乘 (Factorial)** - 经典尾递归案例
2. **求和 (Sum)** - 简单累加
3. **斐波那契 (Fibonacci)** - 带累加器的斐波那契数列
4. **最大公约数 (GCD)** - 欧几里得算法
5. **数组求和** - 递归处理数组
6. **字符串反转** - 字符串操作
7. **Collatz 猜想** - 角谷猜想
8. **数组过滤** - 过滤偶数
9. **数字转二进制** - 进制转换
10. **求幂** - 指数计算

## 预期结果

未优化版本会在大数值时触发栈溢出（RangeError: Maximum call stack size exceeded）。

优化后的版本可以处理：
- 10,000+ 次递归深度
- 100,000+ 次迭代
- 无栈溢出风险
